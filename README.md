# Web Server

## Getting Started

### Dependencies

- Winsock
- http-parser

### Download

```sh
git clone https://github.com/daviskauffmann/web-server.git
cd web-server
```

### Build

```sh
make
```

### Build & Run

```sh
make run
```

### Cleanup

```sh
make clean
```
